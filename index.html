STAT 430 - Introduction to Statistical Computation with R

This very useful applied statistics course provides an introduction to statistical computing with the R programming language. R will be used to execute a large number of statistical procedures and to manipulate large data sets.
In the industry of today, virtually all statisticians work with and manipulate large datasets, sometimes with hundreds of thousands (or even millions) of entries. Imagine trying to open a dataset that large with a program like excel - the application would not even open!

What Is R and Why Are We Using It?

R is a language developed to easily handle loading and manipulating large sets of data. It also contains a ton of "packaged functionality" (called libraries) that make it very easy to perform and automate statistical computation. For example, instead of having to calculate the variance between all elements in a dataset by hand, you can call a function to accomplish the same thing in a single written line!
It is important to keep in mind that R is just a language. We will also make use of the R-Studio IDE (Integrated Data Environment).

What Is R-Studio and What Is An IDE?

Without an IDE, you would have to write the functionality to do statistical analysis in a text file (or even the command line). While this is technically possible to do, it would be very difficult. Imagine performing all of these calculations and modifications without being able to see the changes you are making! Or generating plots and having to write a bunch of complex functionality to determine what file to store the plot as and where to put it!
With R-Studio, you can easily see what data you have loaded, and all manipulations that are taking place on it. It is also very easy to perform various data visualization methods (i.e. displaying plots and graphs). R-Studio does most of the hard work for you, and it also shows the plots on the same screen as the code you wrote.
